# Daily Planner App
A simple daily planner app with email authentication

## Demo
Coming soon

### Stack
MERN: MongoDB, Express, React, Node.js
Redux, Bcrypt, Toastify

## Usage
Rename the .envexample to .env and add your MONGO_URI

### Run the backend
```
npm install
```
```
npm run api
```
### Run the frontend
```
cd client
```
```
npm install
```
```
npm run client
```
### Test e2e in the browser
```
npx cypress:open
```
### Test e2e headless
```
npx cypress:run
```
